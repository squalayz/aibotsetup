# Polish & Refine the Hero Section — Make It Premium

The layout and structure are good. Don't rebuild anything. This is a REFINEMENT pass to make everything feel sleek, premium, and intentional — like an Apple or Linear.app level of polish. Go through each of these changes carefully:

---

## 1. TYPOGRAPHY — Cleaner, More Breathing Room

The headline is too cramped and heavy. Fix it:

- Switch the headline font to `Space Grotesk` (import from Google Fonts) — weight 700 for the headline, 400 for body text. If already using Space Grotesk, switch to `Inter` with weight 800 for headline. The goal is a modern, clean tech font — NOT ultra-condensed or blocky.
- **Reduce the headline font size by about 15-20%.** It's too large right now and feels like it's screaming. It should command attention but feel confident, not loud.
- **Increase line-height on the headline to 1.1** (currently looks like 0.9 or 1.0 which makes it feel compressed)
- **Add letter-spacing: -0.02em** to the headline for a tighter, more premium feel
- Change the headline to: **"We Build AI Agents That Run Your Business"** (change "Empire" back to "Business" — it's more professional and trustworthy)
- The "AI AGENTS" gradient text line is good — keep it but make the gradient smoother (transition from #00e5ff cyan to #a855f7 purple to #22c55e green, using a wider gradient spread)
- **Body text** below the headline: increase font size to 18px, set line-height to 1.7, set color to #9ca3af (slightly brighter gray for readability), and max-width: 520px so lines don't run too wide

## 8. SMOOTH ANIMATIONS — Make Everything Butter

Add these global animation improvements:

### Page Load Sequence (staggered, smooth):
```
0.0s — Nav fades in (opacity 0→1, translateY -10→0)
0.3s — Pill badge fades in (opacity 0→1, translateY 10→0)  
0.5s — Headline line 1 decodes/fades in
0.7s — Headline line 2 ("AI AGENTS") decodes with gradient
0.9s — Headline line 3-4 decode in
1.3s — Body text fades in (opacity 0→1, translateY 15→0)
1.6s — CTA buttons slide in (opacity 0→1, translateX -20→0, staggered 100ms each)
2.0s — Trust bar fades in
2.2s — Scroll indicator appears
0.5s — Robot image fades in from right (opacity 0→1, translateX 40→0) with slight scale 0.95→1
```

All animations should use **cubic-bezier(0.16, 1, 0.3, 1)** easing (smooth, slightly bouncy deceleration — feels premium).

### Scroll Animations:
- Use GSAP ScrollTrigger for ALL below-the-fold sections
- Every section title: AI text decode effect (scrambled characters resolving)
- Every card/element: fade in + translateY 40px → 0 with stagger
- Easing: `power3.out` or `expo.out` in GSAP — never linear, never basic ease-in-out

### Hover Transitions:
- ALL hover effects should be 0.3s with ease-out timing
- No instant/jarring transitions anywhere
- Buttons should have `transform: translateY(-2px)` on hover for a subtle lift effect

- The background should be a very subtle **radial gradient**: `radial-gradient(ellipse at 30% 50%, rgba(0, 229, 255, 0.03) 0%, transparent 60%)` on the left side (where the text is) — this adds a barely-perceptible depth

---

## 10. OVERALL RULES

- **Nothing should feel rushed or sudden** — every transition, every animation should be smooth and intentional
- **Whitespace is your friend** — when in doubt, add more spacing
- **Consistency** — every similar element (all buttons, all cards, all section titles) should use the exact same styles and animation patterns
- **Restraint** — don't add more effects. Polish what's there. Less but better.
- The website should feel like it was designed by a **top-tier design agency**, not generated by AI. The difference is in the small details: consistent spacing, smooth easing curves, thoughtful typography, and visual hierarchy.
