CLAWD Website — Complete Redesign Prompt for Replit
THE VISION
Rebuild the CLAWD website (aibotsetup.com) into an absolutely jaw-dropping, interactive, animated experience that feels like stepping into a living AI command center. This should NOT look like a typical SaaS landing page. It should feel like a sci-fi interface crossed with an art installation — the kind of site people screenshot and share because they've never seen anything like it.
The site is for CLAWD — a business that builds custom AI agents for companies. The brand colors are deep dark navy/black backgrounds with cyan/teal, purple/violet, and green accents.

GLOBAL EFFECTS (Apply across the ENTIRE page)
1. AI Cursor Companions

Spawn 3 small floating AI "eye" orbs (glowing cyan/purple) that follow the user's cursor around the page with smooth spring physics (slight delay, organic movement)
Each orb has a different size and color (teal, purple, green — matching brand)
They should have a subtle "blink" animation every few seconds
When the cursor stops moving, the orbs slowly orbit around the cursor position
When the cursor moves fast, they trail behind with a particle trail effect
On click anywhere, the orbs briefly scatter outward then regroup

2. Neural Network Background

The entire page background should have an animated neural network mesh — interconnected nodes with glowing lines between them
Nodes should pulse and glow randomly, with data packets (small light dots) traveling along the connection lines
The network should subtly react to scroll position — nodes near the viewport center glow brighter
Use a dark navy (#0a0a1a) base with the network rendered on a canvas layer

3. Custom Cursor

Replace the default cursor with a glowing crosshair/ring that pulses
On hover over interactive elements, the ring expands and changes color
Leave a brief fading trail of small dots behind the cursor as it moves

4. Scroll-Triggered Reveals

Every section should animate in with staggered, cinematic reveals — not basic fade-ins
Use combinations of: scale from 0, blur-to-sharp, glitch effects, typewriter text, counter animations
Parallax layers at different scroll speeds for depth

5. Floating Particles

Ambient floating particles across the whole page (tiny glowing dots drifting upward slowly)
These particles should part/scatter away from the cursor as it moves through them (magnetic repulsion effect)

TECHNICAL REQUIREMENTS
Libraries/Frameworks to Use:

GSAP (GreenSock) — for all scroll-triggered animations, timelines, and complex tweens
Three.js — for the 3D neural sphere in the hero and any 3D elements
Canvas API — for the neural network background and particle systems
Framer Motion (if using React) — for component animations
Intersection Observer — for scroll-triggered reveals
CSS Custom Properties — for dynamic theming and cursor position variables

Performance:

Use will-change and GPU-accelerated properties (transform, opacity) for all animations
Implement requestAnimationFrame for canvas animations
Lazy load sections below the fold
Use prefers-reduced-motion media query to disable heavy animations for accessibility
Target 60fps on all animations

Responsive:

All animations should gracefully degrade on mobile
AI cursor companions should be disabled on touch devices (no cursor)
Neural network background should reduce node count on mobile
Cards should stack vertically on mobile with simplified animations

Color Palette:

Background: #0a0a1a (deep navy/black)
Primary: #00e5ff (cyan/teal)
Secondary: #a855f7 (purple/violet)
Accent: #22c55e (green)
Text: #ffffff (white) and #94a3b8 (muted gray)
Card backgrounds: rgba(255, 255, 255, 0.05) with backdrop-blur

Typography:

Headlines: Bold, modern sans-serif (Inter, Space Grotesk, or similar)
Body: Clean, readable sans-serif
Code/status text: Monospace font for that techy feel


THE VIBE CHECK
This website should make someone say "holy shit" when they land on it. It should feel alive — like the AI agents are already watching and ready to work. Every scroll, every hover, every click should trigger something delightful and unexpected. The animations should be smooth, purposeful, and layered — not chaotic. Think: Stripe's homepage meets a sci-fi movie interface meets an interactive art piece.
